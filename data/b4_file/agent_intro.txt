Agent 基础介绍文档
1. Agent 定位
本智能 Agent 是具备自主规划、工具调用、多轮推理、任务拆解执行能力的通用智能执行主体，依托大语言模型作为核心大脑，可自动识别用户复杂需求，拆分子步骤，调用外部工具完成检索、计算、代码运行、文件处理、数据分析等操作，闭环完成完整业务任务，无需人类分步引导。
2. 核心能力
需求解析与任务拆解
接收自然语言复杂指令，识别目标、约束条件、输出格式要求，将长任务拆分为有序可执行子任务，生成执行计划 Plan。
自主工具调度（Tool Calling）
内置工具调度逻辑，自动判断是否需要调用工具；支持检索、计算器、代码解释器、文件读写、图片处理、联网搜索等工具，自动拼装工具入参、执行、接收返回结果。
多轮反思修正
工具返回异常、执行结果不满足需求、逻辑存在漏洞时，自动反思错误原因，调整执行方案重新执行，具备自纠错能力。
上下文记忆管理
持久保存对话历史、中间计算数据、临时文件信息，区分短期会话记忆与长期知识库记忆，按需调取历史信息辅助推理。
结构化输出控制
支持自定义输出格式：纯文本、Markdown、表格、JSON、LaTeX、代码片段、标准报告等，严格遵循用户指定格式返回结果。
多领域适配
覆盖算法竞赛、LLM 评测、系统开发、学术文书、数据分析、保研规划、文档处理、数学建模、网络底层学习等技术场景。
